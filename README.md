# AgriProject – Agricultural Data Analytics using Python, SQL & ML

Comprehensive Crop Production Analysis of India (1950–2020)

## Project Overview

The AgriProject aims to analyze agricultural production trends across India using Python (EDA), SQL, and Machine Learning. It explores multi-crop data (Rice, Wheat, Maize, Millets, Oilseeds, etc.), performs statistical and visual analysis, and builds predictive models for crop yield forecasting.
